# Community

OpenTalking is developed as an open-source project. Contributions, feedback, and
discussion are welcome through the channels below.

## Channels

- **GitHub** — [datascale-ai/opentalking](https://github.com/datascale-ai/opentalking)
- **Issues and feature requests** — [GitHub Issues](https://github.com/datascale-ai/opentalking/issues)
- **QQ group** — group number `1103327938` (AI 数字人交流群). The QQ group is the
  primary Chinese-language venue for discussion of real-time digital humans,
  FlashTalk, OmniRT, and deployment topics.

![QQ group QR code](../../assets/images/qq_group_qrcode.png){ width=280 }

## Contributing

Pull requests of any size are welcome. Before submission, review the
[Contributing](../developer-guide/contributing.md) guide for the project's
conventions on interface-first design, commit scope, and documentation alongside
code. Draft pull requests opened early in the development cycle allow reviewers to
provide direction.

## Roadmap

Project planning and release history are tracked on GitHub:

- [Issues](https://github.com/datascale-ai/opentalking/issues) — bug reports and feature requests.
- [Releases](https://github.com/datascale-ai/opentalking/releases) — version history and changelog.
- [Discussions](https://github.com/datascale-ai/opentalking/discussions) — design proposals and questions.
