# 社区

OpenTalking 以开源项目形式开发。欢迎通过以下渠道参与贡献、反馈与讨论。

## 渠道

- **GitHub** —— [datascale-ai/opentalking](https://github.com/datascale-ai/opentalking)
- **Issue 与功能请求** —— [GitHub Issues](https://github.com/datascale-ai/opentalking/issues)
- **QQ 群** —— 群号 `1103327938`（AI 数字人交流群）。该群为实时数字人、FlashTalk、
  OmniRT 与部署相关话题的主要中文讨论场所。

![AI 数字人交流群二维码](../../assets/images/qq_group_qrcode.png){ width=280 }

## 参与贡献

欢迎各种规模的 pull request。提交之前请阅读 [贡献指南](../developer-guide/contributing.md)
中关于接口优先设计、commit 范围、代码与文档同步的约定。在开发周期早期开 Draft PR，
便于审阅者提前给出方向性反馈。

## Roadmap

项目规划与发布历史均在 GitHub 上维护：

- [Issues](https://github.com/datascale-ai/opentalking/issues) —— bug 报告与功能请求。
- [Releases](https://github.com/datascale-ai/opentalking/releases) —— 版本历史与 changelog。
- [Discussions](https://github.com/datascale-ai/opentalking/discussions) —— 设计提案与问答。
